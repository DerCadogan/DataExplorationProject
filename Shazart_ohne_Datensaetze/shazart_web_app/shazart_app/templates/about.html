Welcome to Shazart

Shazart is a free-to-use service that can tell you within seconds which art epochs style a picture has.

This is done through a Convolutional Neurol Network (CNN).

The Neural Network can distinguish between 5 different Epochs:

  1. Bauhaus
  2. Expressionism
  3. Impressionism
  4. Renaissance
  5. Romantic
